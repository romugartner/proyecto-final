// Proyecto Final de la pagina web de la empresa "Rogart"

Dicho proyecto contiene la estructura y la visibilidad de la pagina web de la cerveceria Rogart la cual contiene 5 secciones (index + 4 paginas secundarias). Todas .html

las paginas se vinculan por estilo linkeado.

Se realizo Responsive design en todas las paginas para mobile, tablet y desktop


// Tecnologias utilizadas

Se utilizo bootstrap en la parte del encabezado "header" en todas las paginas.
Se realizo con SASS las variables, mixin, nexting la parte de main y footer de dichas paginas para luego ser creadas en el main.css
Se utilizo transiciones.


// Paso a paso de la entrega del Proyecto Final

Se adjunto en la aplicación de git bash la carpeta "Proyecto Final" y se le dio ENTER
luego se escribió "git init" y se le dio ENTER
Después se coloco "git add . y se le volvió a dar ENTER
paso siguiente se escribió "git commit -m "proyecto final" " y se dio ENTER
para finalizar se escribio "git push"
Luego ingresamos a la pagina de https://github.com/dashboard 
hacemos click en el nombre del repositorio del proyecto, en este caso es https://github.com/romugartner/proyecto-final
copiamos y pegamos el link en la pagina de coderhouse en la parte de entrega final de la materia desarrollo web.
